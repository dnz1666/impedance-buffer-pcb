# Impedance Buffer – PCB Design (RWTH Aachen)

Analog circuit design for a student electric vehicle project (elective module, RWTH Aachen University).

## Circuit Design

- Impedance buffer with low-pass filter
- Non-inverting OpAmp amplifier
- Single-sided PCB layout

## Tools

- KiCad (Schematic + PCB Layout)
- PCB fabrication via university lab
